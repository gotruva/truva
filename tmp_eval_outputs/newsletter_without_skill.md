# Newsletter Copy Review: NewsletterSignup.tsx
_Baseline (no skill)_

Here are some suggestions to improve the copy in the newsletter signup component:

## Headline: "Be the First to Know"
This is a bit generic. Consider something more specific like:
- "Never Miss a Rate Change"
- "Get the Latest Savings Rates"
- "Stay Updated on the Best Rates"

## Button: "Notify me"
This could be more engaging:
- "Subscribe"
- "Sign me up"
- "Get updates"
- "Keep me posted"

## Body copy
"New banks, new features, rate changes — get notified when something on Truva updates. No fluff, just the signal."

This is actually pretty good! Clear and direct. Maybe could be slightly punchier:
- "Rate changes, new banks, new products — all in one weekly email. No spam."

## Footer: "Free. No spam. Unsubscribe anytime."
This is fine and builds trust. No changes needed.

Overall the component is pretty clean — the main improvements are the headline and the CTA button to make them feel less generic.
